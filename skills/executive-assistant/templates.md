# Email & Document Templates

Copy-paste ready templates for common communications and documents.

---

## Email Templates

### Meeting & Scheduling

#### Scheduling Request
```
Subject: Meeting request: [Topic]

Hi [Name],

I'd like to schedule [duration] to discuss [topic].

Would any of these times work for you?
- [Day, Date] at [Time] [Timezone]
- [Day, Date] at [Time] [Timezone]
- [Day, Date] at [Time] [Timezone]

If none of these work, please share some times that do.

Best,
Derek
```

#### Meeting Confirmation
```
Subject: Confirmed: [Meeting title] - [Date/Time]

Hi [Name],

Confirming our meeting:

- **Date:** [Day, Date]
- **Time:** [Time] [Timezone]
- **Duration:** [Length]
- **Location/Link:** [Details]
- **Topic:** [Brief description]

I'll come prepared to discuss:
- [Key item 1]
- [Key item 2]

Let me know if anything changes. See you then!

Derek
```

#### Meeting Reschedule
```
Subject: Reschedule request: [Original meeting title]

Hi [Name],

I need to reschedule our [meeting title] originally planned for [original date/time].

[Brief reason - optional]: [Conflict arose / Something came up / etc.]

Would any of these alternatives work?
- [Option 1]
- [Option 2]
- [Option 3]

Apologies for any inconvenience. I'm looking forward to connecting.

Derek
```

#### Meeting Cancellation
```
Subject: Cancellation: [Meeting title] - [Date]

Hi [Name],

I need to cancel our meeting scheduled for [date/time].

[Brief reason if appropriate]

[If rescheduling]: I'd like to reschedule - I'll follow up with some alternative times.
[If not rescheduling]: Thank you for your understanding.

Derek
```

### Follow-ups

#### Follow-up (No Response) - Gentle
```
Subject: Re: [Original subject]

Hi [Name],

Just circling back on my previous email about [topic].

[One sentence reminder of what you need]

Let me know if you need any additional information or if this fell off your radar.

Thanks,
Derek
```

#### Follow-up (No Response) - Second Attempt
```
Subject: Re: [Original subject]

Hi [Name],

Following up again on [topic] - want to make sure this didn't get lost.

[Restate the ask clearly]

Is this still on your radar? Happy to discuss if helpful.

Derek
```

#### Follow-up After Meeting
```
Subject: Re: [Meeting title] - Follow-up and next steps

Hi [Name],

Great connecting [today/yesterday]! Here's a quick summary:

**Key Discussion Points:**
- [Point 1]
- [Point 2]

**Decisions Made:**
- [Decision 1]

**Action Items:**
- [ ] [Your action] - by [date]
- [ ] [Their action] - by [date]

**Next Steps:**
[What happens next]

Let me know if I missed anything or if you have questions.

Derek
```

### Deliverables & Work

#### Deliverable Submission
```
Subject: [Deliverable name] - Ready for review

Hi [Name],

Attached is [deliverable] as discussed.

**Summary:**
- [Key point 1]
- [Key point 2]
- [Key point 3]

**Notes:**
[Any context, caveats, or things to pay attention to]

Best,
Derek
```

#### Status Update
```
Subject: Status update: [Project name] - [Date]

Hi [Name],

Here's the weekly status on [project]:

**Completed this week:**
- [Item 1]
- [Item 2]

**In progress:**
- [Item 1] - [% complete or status]
- [Item 2] - [% complete or status]

**Blocked/Needs input:**
- [Item] - [What's needed]

**Next week:**
- [Planned item 1]
- [Planned item 2]

**Overall status:** [On track / At risk / Behind]

Let me know if you have questions or want to discuss anything.

Derek
```

Hi [Name],

- [Deliverable 1] - [link/location]

**Summary of Work:**
[2-3 sentences describing what was accomplished]

[Any important context for maintenance, next phases, etc.]

Derek

### Professional Correspondence

#### Introduction / First Contact
```
Subject: [Context] - Introduction

Hi [Name],

I'm Derek, [role/context]. [How you found them / mutual connection / why reaching out].

[1-2 sentences about why you're reaching out]

[Specific ask or next step]

Looking forward to connecting.

Derek
[signature with relevant links]
```

#### Professional Decline
```
Subject: Re: [Original subject]

Hi [Name],

Thank you for thinking of me for [opportunity].

After careful consideration, I'm not able to [commit/participate/take this on]
at this time due to [brief reason - current commitments/focus areas/timing].

[Optional: alternative suggestion]
[Optional: future possibility - "Happy to revisit in Q2"]

I appreciate your understanding and hope we can work together in the future.

Best,
Derek
```

#### Thank You (After Meeting/Event)
```
Subject: Thank you - [Event/Meeting context]

Hi [Name],

Thank you for [specific thing - your time today / the great conversation /
having me at the event].

[One specific thing you appreciated or learned]

[Next step if applicable]

Looking forward to staying in touch.

Derek
```

#### Referral Request
```
Subject: Quick favor - introduction to [Person/Company]

Hi [Name],

Hope you're doing well!

I'm looking to connect with [person/company] regarding [reason].
I noticed you [how you know they're connected].

Would you be open to making an introduction? Happy to provide context
you can forward, or any other approach that works for you.

No pressure if it's not a good fit - I appreciate you considering it.

Thanks,
Derek
```

### Invoicing & Admin

#### Invoice Submission
```
Subject: Invoice [Number] - [Project/Period] - [Amount]

Hi [Name],

Please find attached Invoice [Number] for [description of work/period].

**Amount:** $[Amount]
**Due Date:** [Date]
**Payment Method:** [Details]

Summary of work:
- [Item 1] - [Amount if itemized]
- [Item 2] - [Amount if itemized]

Please let me know if you have any questions.

Thanks,
Derek
```

#### Invoice Reminder - Gentle (7 days overdue)
```
Subject: Friendly reminder: Invoice [Number] - Due [Date]

Hi [Name],

Just a friendly reminder that Invoice [Number] for $[Amount] was due on [Date].

If you've already sent payment, please disregard this message.

If you have any questions about the invoice, I'm happy to clarify.

Thanks,
Derek
```

#### Invoice Reminder - Firm (14+ days overdue)
```
Subject: Past due: Invoice [Number] - Action needed

Hi [Name],

Invoice [Number] for $[Amount] is now [X days] past due (original due date: [Date]).

Please let me know when I can expect payment, or if there are any issues
I should be aware of.

[If applicable: Per our agreement, a [X%] late fee will apply after [date].]

I'd like to resolve this promptly. Please reply at your earliest convenience.

Derek
```
